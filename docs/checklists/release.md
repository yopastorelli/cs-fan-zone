# Release Checklist

- [ ] Experience remains private until compliance review is complete.
- [ ] No real names, logos, voices, screenshots, copied slogans, or recognizable avatars without license.
- [ ] No off-platform links, QR codes, Discord prompts, email requests, or personal-data collection.
- [ ] `docs/asset-register.md` lists every external asset.
- [ ] `docs/ATTRIBUTIONS.md` includes all required credits.
- [ ] `rojo build -o build.rbxlx` succeeds.
- [ ] Studio Output has no red errors after 2 minutes of play.
- [ ] HUD is legible on desktop and mobile viewport.
- [ ] First collectible is reachable in under 60 seconds.
- [ ] Full MVP loop can be completed in 5 to 10 minutes.

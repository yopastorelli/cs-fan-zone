# AI_LICENSE_EVIDENCE_SPEC

PURPOSE: Store proof that external assets can be used in this Roblox experience.
CURRENT_CONTENT_REQUIRED: none

## ACCEPTED_EVIDENCE
- License text snapshot.
- Source page screenshot.
- Purchase receipt.
- Written permission.

## REJECTION_RULE
If license evidence is absent, unclear, noncommercial only, share-alike, GPL, or conflicts with interactive game use, do not use the asset.

# AI_ASSETS_SPEC

PURPOSE: Store local source files, Roblox-ready imports, and license evidence.
CURRENT_EXTERNAL_ASSETS: none
DEFAULT_ASSET_POLICY: use procedural Roblox Parts and built-in materials.

## DIRECTORIES
raw: original downloaded or project-created source files before processing.
imported: processed files prepared for Roblox Studio import.
licenses: screenshots, text files, receipts, or license snapshots proving terms.

## REQUIRED_BEFORE_IMPORT
1. Add asset row to docs/asset-register.md.
2. Add attribution row to docs/ATTRIBUTIONS.md when license requires credit.
3. Store license evidence under assets/licenses.
4. Reject asset if source, author, license, or permitted use is unclear.

## BLOCKED_ASSETS
- Fandom reuploads.
- Unofficial mirrors.
- Logos, real people, recognizable likenesses, voices, screenshots, copied slogans, or protected symbols.
- NC, SA, GPL, unclear, or mixed-license files.
- Any asset containing off-platform links, QR codes, emails, or social prompts.
